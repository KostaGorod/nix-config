# rocinante

Daily-driver NixOS workstation.

## Hardware

| Component | Value                                        |
|-----------|----------------------------------------------|
| Model     | Lenovo ThinkPad X1 Carbon Gen 9              |
| CPU       | 11th Gen Intel Core i7-1165G7 (8 threads)    |
| GPU       | Intel Iris Xe (integrated)                   |
| Memory    | 16 GiB                                       |
| Display   | CMN1417, 1920×1200 @ 60 Hz, 14"              |
| Security  | TPM 2.0, fingerprint reader, YubiKey (FIDO2) |

## Files in this directory

| File                       | Purpose                                              |
|----------------------------|------------------------------------------------------|
| `default.nix`              | Host-wide hardware, boot, networking, nix settings   |
| `configuration.nix`        | Host-specific services (TLP, printing, mem0, …)      |
| `hardware-configuration.nix` | Generated by `nixos-generate-config`               |
| `disko-config.nix`         | Declarative disk layout (LUKS + btrfs)               |

Host-orthogonal concerns live one level up: desktop environments in `de/`,
AI tooling + editors in `profiles/workstation.nix`, reusable services in
`modules/nixos/`.

## First-time deploy

1. Boot NixOS installer, partition with disko:
   ```sh
   sudo nix run github:nix-community/disko -- \
     --mode disko ./disko-config.nix
   ```
2. Install:
   ```sh
   sudo nixos-install --flake .#rocinante
   ```
3. Boot into the new system, then bootstrap secrets:
   - Copy `/etc/ssh/ssh_host_ed25519_key.pub` into the repo's
     `secrets.nix` under `rocinante =`.
   - `nix run github:ryantm/agenix -- --rekey`
   - Rebuild once more so `/run/secrets/*` populates.

Day-to-day:

```sh
sudo nixos-rebuild switch --flake .#rocinante
```

## Notable setup

- **Hardware-backed SSH** via `security.ssh-tpm` (see
  `modules/nixos/ssh-tpm-pkcs11.nix`). Keys live in the TPM; FIDO2-resident
  keys on YubiKey are prompted via lxqt-openssh-askpass.
- **Dual desktop environments.** Plasma 6 and COSMIC are both installed and
  selectable at login.
- **TLP tuned per-profile** with battery-health thresholds (40/80) and
  runtime PM denylist for the touchpad controller.
- **Tailscale + dnsmasq split-DNS** via `services.tailscale-mesh.enable`.
